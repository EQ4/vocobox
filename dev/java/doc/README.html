INTRO

A live voice analyser to control digital musical instruments.

SINGER note, amplitude, formant -> SYNTHETISER CONTROLER -> SYNTHETIZER: midi, emulated

The first Vocobox implementation provide

  • a full java additive synthetizer made by chaining oscillators, filters, and other hardware emulation of real synthetizer components. It is build JSyn, a wonderfull Java Synthetizer framework written by Phil Burk.
  • a pitch detection module based on the excellent TarsosDSP java library written by Jochen Six.
  • charts for monitoring and debugging the sound data workflow.

A future version will come with a MIDI emitter
